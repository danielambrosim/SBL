## 🚀 Instalação

```bash
# Clone o repositório
git clone https://github.com/seuusuario/bot-leilao.git && cd bot-leilao

# Instale as dependências
npm install

# Configure (renomeie e edite)
cp .env.example .env
O bot irá iniciar e ficará disponível para interação no Telegram!
